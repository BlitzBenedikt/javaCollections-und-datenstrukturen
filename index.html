Java Collections und Datenstrukturen

In dieser Lektion lernst du die wichtigsten Datenstrukturen und Collections in Java kennen. Collections sind Sammlungen von Objekten, die es dir ermöglichen, Daten effizient zu speichern, zu organisieren und zu manipulieren.

  1. Listen: ArrayList und LinkedList
  2. Sets: HashSet und TreeSet
  3. Maps: HashMap und TreeMap
  4. Wichtige Collection-Methoden
  5. Fortgeschrittene Aufgaben

1. Listen: ArrayList und LinkedList

Listen sind geordnete Collections, in denen Elemente doppelt vorkommen können. Es gibt zwei Hauptimplementierungen: ArrayList und LinkedList.

ArrayList LinkedList
Basiert auf einem dynamischen Array Basiert auf einer verketteten Liste
Schneller Zugriff auf Elemente per Index Langsamer Zugriff auf Elemente per Index
Langsamer beim Einfügen/Löschen in der Mitte Schneller beim Einfügen/Löschen in der Mitte 
// ArrayList erstellen
ArrayList<String> namenListe = new ArrayList<>();

// Elemente hinzufügen
namenListe.add("Anna");
namenListe.add("Ben");
namenListe.add("Clara");

// Element an bestimmter Position einfügen
namenListe.add(1, "David");  // Fügt "David" an Position 1 ein

// Auf Elemente zugreifen
String ersterName = namenListe.get(0);  // "Anna"

// Elemente entfernen
namenListe.remove("Ben");  // Entfernt "Ben"
namenListe.remove(0);      // Entfernt erstes Element

// Größe einer Liste
int anzahl = namenListe.size();

// Prüfen, ob ein Element enthalten ist
boolean enthalten = namenListe.contains("Clara");  // true

// Durch eine Liste iterieren
for (String name : namenListe) {
    System.out.println(name);
}

Schritt-für-Schritt: Erstellen und Verwenden einer ArrayList

  1. Importiere die ArrayList-Klasse: import java.util.ArrayList;
  2. Erstelle eine ArrayList für den gewünschten Datentyp: ArrayList<Typ> listenName = new ArrayList<>();
  3. Füge Elemente hinzu mit add()
  4. Greife auf Elemente zu mit get(index)
  5. Aktualisiere Elemente mit set(index, neuesElement)
  6. Entferne Elemente mit remove()

2. Sets: HashSet und TreeSet

Sets sind Collections, die keine doppelten Elemente enthalten. Die zwei wichtigsten Implementierungen sind:

HashSet TreeSet
Unsortierte Menge von Elementen Sortierte Menge von Elementen
Sehr schnell bei allen Operationen Etwas langsamer, aber sortiert 
// HashSet erstellen
HashSet<Integer> zahlen = new HashSet<>();

// Elemente hinzufügen
zahlen.add(5);
zahlen.add(10);
zahlen.add(15);
zahlen.add(5);  // Wird ignoriert, da 5 bereits enthalten ist

System.out.println(zahlen);  // [5, 10, 15]

// TreeSet für sortierte Elemente
TreeSet<String> sortierteNamen = new TreeSet<>();
sortierteNamen.add("Max");
sortierteNamen.add("Anna");
sortierteNamen.add("Ben");

System.out.println(sortierteNamen);  // [Anna, Ben, Max]

Schritt-für-Schritt: Arbeiten mit Sets

  1. Importiere die Set-Klassen: import java.util.HashSet; oder import java.util.TreeSet;
  2. Erstelle ein Set: HashSet<Typ> setName = new HashSet<>();
  3. Füge Elemente hinzu mit add()
  4. Prüfe Enthaltensein mit contains()
  5. Entferne Elemente mit remove()
  6. Iteriere durch das Set mit einer for-each Schleife

3. Maps: HashMap und TreeMap

Maps speichern Schlüssel-Wert-Paare. Jeder Schlüssel ist eindeutig und mit einem Wert verknüpft.

HashMap TreeMap
Unsortiert Nach Schlüsseln sortiert
Schnellere Operationen Etwas langsamer, aber sortiert 
// HashMap erstellen
HashMap<String, Integer> altersMap = new HashMap<>();

// Schlüssel-Wert-Paare hinzufügen
altersMap.put("Anna", 25);
altersMap.put("Ben", 30);
altersMap.put("Clara", 22);

// Wert über Schlüssel abrufen
int annasAlter = altersMap.get("Anna");  // 25

// Prüfen, ob ein Schlüssel existiert
boolean benExistiert = altersMap.containsKey("Ben");  // true

// Durch die Map iterieren
for (String name : altersMap.keySet()) {
    System.out.println(name + " ist " + altersMap.get(name) + " Jahre alt.");
}

// Alle Werte durchlaufen
for (int alter : altersMap.values()) {
    System.out.println("Alter: " + alter);
}

// Schlüssel-Wert-Paare durchlaufen
for (Map.Entry<String, Integer> eintrag : altersMap.entrySet()) {
    System.out.println(eintrag.getKey() + ": " + eintrag.getValue());
}

Schritt-für-Schritt: Arbeiten mit Maps

  1. Importiere die Map-Klassen: import java.util.HashMap; oder import java.util.TreeMap;
  2. Erstelle eine Map: HashMap<Schlüsseltyp, Werttyp> mapName = new HashMap<>();
  3. Füge Schlüssel-Wert-Paare hinzu mit put(schlüssel, wert)
  4. Greife auf Werte zu mit get(schlüssel)
  5. Prüfe, ob ein Schlüssel existiert mit containsKey(schlüssel)
  6. Entferne Einträge mit remove(schlüssel)

4. Wichtige Collection-Methoden

Die Collections-Klasse bietet nützliche statische Methoden für Collections:

import java.util.Collections;
import java.util.ArrayList;
import java.util.List;

// Liste erstellen und befüllen
List<Integer> zahlen = new ArrayList<>();
zahlen.add(3);
zahlen.add(1);
zahlen.add(5);
zahlen.add(2);

// Sortieren
Collections.sort(zahlen);
System.out.println("Sortiert: " + zahlen);  // [1, 2, 3, 5]

// Maximum und Minimum finden
int max = Collections.max(zahlen);  // 5
int min = Collections.min(zahlen);  // 1

// Liste umkehren
Collections.reverse(zahlen);
System.out.println("Umgekehrt: " + zahlen);  // [5, 3, 2, 1]

// Zufällige Anordnung
Collections.shuffle(zahlen);
System.out.println("Gemischt: " + zahlen);

// Häufigkeit eines Elements
int anzahl = Collections.frequency(zahlen, 3);  // Wie oft kommt 3 vor?

Schritt-für-Schritt: Verwenden der Collections-Klasse

  1. Importiere die Collections-Klasse: import java.util.Collections;
  2. Erstelle und befülle eine Collection (z.B. ArrayList)
  3. Nutze die statischen Methoden von Collections, z.B. Collections.sort(liste)
  4. Weitere nützliche Methoden: max(), min(), reverse(), shuffle()
